# Output Formats

Specifications for each deliverable generated by the resume workflow.

---

## 1. Resume

**Format:** Markdown → DOCX (via pandoc)

**Pandoc command:**
```bash
pandoc [Name]_Resume_FINAL.md -o [Name]_Resume_FINAL.docx --reference-doc="templates/resume_template.docx"
```

**File naming:**
- Draft: `[Name]_Resume_DRAFT.md` / `.docx`
- Final: `[Name]_Resume_FINAL.md` / `.docx`

**Structure:** See `system/resume_rules.md` for full structure and formatting rules.

---

## 2. Interview Prep Guide

**Format:** Markdown → DOCX (via pandoc)

**Pandoc command:**
```bash
pandoc [Name]_Interview_Prep.md -o [Name]_Interview_Prep.docx --reference-doc="templates/resume_template.docx"
```

**Content:** Cover the 5-7 strongest resume bullets. For each:

```markdown
### [Achievement Title]

**Resume Bullet:**
> [The exact bullet from the resume]

**STAR Breakdown:**

- **Situation:** [Context — what was happening, what was the problem]
- **Task:** [What you were responsible for, what needed to happen]
- **Action:** [Specific steps you took — be detailed]
- **Result:** [Quantified outcome — use the same metrics as the resume]

**Follow-Up Questions to Expect:**
- [Likely follow-up question 1]
  - [How to answer]
- [Likely follow-up question 2]
  - [How to answer]

**"What would you do differently?"**
[Honest, thoughtful answer that shows self-awareness without undermining the achievement]
```

## 3. Skills Matrix

**Format:** HTML with embedded CSS

**File naming:** `[Name]_Skills_Matrix.html`

**Color coding:**

| Level | Color | Hex |
|-------|-------|-----|
| Expert | Dark Blue | #3d5a80 |
| Advanced | Gold | #e9c46a |
| Intermediate | Green | #52b788 |
| Novice | Coral | #e76f51 |

**HTML template:**

```html
<!DOCTYPE html>
<html>
<head>
<style>
  body { font-family: 'Segoe UI', Arial, sans-serif; max-width: 800px; margin: 40px auto; padding: 20px; }
  h1 { text-align: center; margin-bottom: 5px; }
  h2 { text-align: center; color: #666; font-weight: normal; margin-top: 0; }
  h3 { margin-top: 30px; margin-bottom: 10px; border-bottom: 2px solid #333; padding-bottom: 5px; }
  table { width: 100%; border-collapse: collapse; margin-bottom: 20px; }
  td { padding: 8px 12px; border-bottom: 1px solid #eee; }
  td:first-child { width: 70%; }
  td:last-child { width: 30%; text-align: center; font-weight: bold; border-radius: 4px; }
  .expert { background-color: #3d5a80; color: white; }
  .advanced { background-color: #e9c46a; color: #333; }
  .intermediate { background-color: #52b788; color: white; }
  .novice { background-color: #e76f51; color: white; }
  .legend { display: flex; gap: 20px; justify-content: center; margin: 20px 0 30px; flex-wrap: wrap; }
  .legend-item { display: flex; align-items: center; gap: 6px; }
  .legend-swatch { width: 16px; height: 16px; border-radius: 3px; }
</style>
</head>
<body>

<h1>Skills Matrix</h1>
<h2>[Candidate Name]</h2>

<div class="legend">
  <div class="legend-item"><div class="legend-swatch" style="background:#3d5a80"></div> Expert</div>
  <div class="legend-item"><div class="legend-swatch" style="background:#e9c46a"></div> Advanced</div>
  <div class="legend-item"><div class="legend-swatch" style="background:#52b788"></div> Intermediate</div>
  <div class="legend-item"><div class="legend-swatch" style="background:#e76f51"></div> Novice</div>
</div>

<!-- Repeat this block for each category group -->
<h3>[Category Group Name]</h3>
<table>
  <tr><td>[Skill Name]</td><td class="[level]">[Level]</td></tr>
  <!-- ... more rows ... -->
</table>

</body>
</html>
```

**Rules:**
- Only include skills relevant to the candidate
- Omit categories rated as Novice unless the candidate specifically wants to show them
- Group skills using the categories from the selected industry profile
- Categories and skills come from the profile, not hardcoded
