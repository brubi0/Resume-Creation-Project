# Output Folder

Generated files will be saved here.

**Expected outputs per candidate:**
1. `[Name]_Resume.docx` — A+ transformed resume
2. `[Name]_Interview_Prep.docx` — STAR-format answers for key achievements
3. `[Name]_Skills_Matrix.html` or `.pdf` — Visual proficiency breakdown

**Naming convention:**
`[FirstName]_[LastName]_[Document].[ext]`

Example:
- `John_Smith_Resume.docx`
- `John_Smith_Interview_Prep.docx`
- `John_Smith_Skills_Matrix.html`
