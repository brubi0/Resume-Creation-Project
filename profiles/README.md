# Profiles

Industry/role profiles that define the skills matrix categories and proficiency levels for a target role.

## Existing Profiles

| Profile | Target Roles |
|---------|-------------|
| `netsuite_administrator.md` | NetSuite Admin, NetSuite Consultant, ERP Analyst, Controller |

## Creating a New Profile

Profiles are auto-generated during Phase 0 of the resume workflow. Claude will:

1. Ask about the candidate's industry and target role
2. Research current job postings to identify required skills
3. Group skills into categories with proficiency definitions
4. Save the profile here for reuse

## Profile Format

Each profile contains:
- Industry and target roles
- Proficiency level definitions (with colors for the skills matrix)
- Skills organized by category with descriptions of what each skill covers
